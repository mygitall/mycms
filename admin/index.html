数据库配置
填写应用将要使用的数据库名与用户名（可以填尚不存在的名字）。保存时程序会尽量用 MySQL 管理员（root）自动建库。

虚拟主机用户：

  1. 登录主机控制面板（如 cPanel、DirectAdmin 等）
  2. 找到「数据库」或「MySQL 数据库」管理功能
  3. 创建或查看已有的数据库、用户名及密码
  4. 数据库地址通常填 localhost，端口填 3306

本地开发环境用户：

  1. 在 phpStudy/MAMP 面板中查看 root 用户名和密码
  2. 常见默认密码：root、xiaopi、root（无密码）

App 接入步骤
1
调用登录/注册接口
App 启动后调用登录接口，code=0 表示登录成功，code=401 表示失败需重新登录。
// Android / Java URL url = new URL("YOUR_DOMAIN/api/login"); HttpURLConnection conn = (HttpURLConnection) url.openConnection(); conn.setRequestMethod("POST"); conn.setRequestProperty("Content-Type", "application/json"); conn.setDoOutput(true); String body = new ObjectMapper().writeValueAsString( Map.of("username","zhangsan","password","123456","device","iPhone15-ABC") ); conn.getOutputStream().write(body.getBytes()); int code = conn.getResponseCode(); String resp = new String(conn.getInputStream().readAllBytes()); // resp 即 JSON 响应字符串
2
本地存储 Token
登录成功后将 token 存入本地，App 关闭后重新打开直接读取，无需重复登录。
// Android — 存 SharedPreferences SharedPreferences sp = getSharedPreferences("app", MODE_PRIVATE); sp.edit().putString("auth_token", token).apply(); // iOS — 存 Keychain（推荐）或 UserDefaults Keychain.save("auth_token", token); // 或简单版： UserDefaults.standard.set(token, forKey: "auth_token")
3
后续请求带上 Token
调用其他接口（查看数据、提交信息等）时，在请求头中加入 Authorization。
// Android conn.setRequestProperty("Authorization", "Bearer " + token); // iOS request.setValue("Bearer \(token)", forHTTPHeaderField: "Authorization") // Flutter headers: {'Authorization': 'Bearer $token'}
4
Token 过期处理
当服务器返回 code=401 时，说明 Token 无效或已过期，清除本地 Token，跳转到登录页面让用户重新登录。
// Android — 拦截器中统一处理 if (response.code == 401) { sp.edit().remove("auth_token").apply(); startActivity(new Intent(this, LoginActivity.class)); finish(); }
文章模块 · 基础信息v1.0.1
完整地址格式{base}/article/api/list
（{base} = 当前部署路径，如 http://localhost/cms，自动适配任意部署位置）
模块基础路径{base}/article/api
请求方式POST（必须用 POST，GET 无效）
Content-Typeapplication/json
认证list 为公开接口，无需认证。
写操作（create/update/delete/favorite）需认证：Authorization: Bearer <token>，或在 JSON Body 中带 _token
CSRF写操作需通过 CSRF 校验（同源表单自动放行，跨域请求请先 GET {base}/api/csrf_token 取 Cookie，请求头带 X-CSRF-Token）
频率限制写操作统一限流：每分钟最多 10 次创建、20 次更新/删除、30 次收藏/取消收藏，超限返回 429 + Retry-After 头
POST {base}/article/api/list 文章列表（分页与筛选）
请求参数（JSON Body）
{ "page": 1, // number，可选，默认 1 "page_size": 10, // number，可选，1–100，默认 10 "keyword": "", // string，可选，标题模糊搜索 "category": "", // string，可选，精确分类 "status": "", // string，可选，"0" 草稿 / "1" 已发布 "start_date": "", // string，可选，YYYY-MM-DD "end_date": "", // string，可选，YYYY-MM-DD "sort_by": "id", // string，可选：id|title|view_count|published_at|created_at|updated_at "sort_order": "desc" // string，可选，asc / desc }
成功响应（节选）
{ "code": 0, "msg": "success", "data": { "list": [ { "id", "title", "content", "category", "tags", ... } ], "categories": ["分类A", "分类B"], "total": 100, "page": 1, "page_size": 10, "total_pages": 10 } }

{ "name": "Visual Studio Code", // string，必填，≤200字符，软件名称 "version": "v1.85.0", // string，必填，版本号如 v1.0.1 "download_urls": "https://...", // string，必填，下载地址，支持多地址用\n换行分隔 "category_name": "开发工具", // string，可选，分类名称（不存在则自动创建） "os_support": "Windows,macOS", // string，可选，逗号分隔，可选值：Windows / macOS / Linux / Android / iOS "file_size": "98.5", // string，可选，软件大小，单位 MB "screenshots": "https://...", // string，可选，多图URL，逗号分隔 "description": "<p>介绍...</p>", // string，可选，软件介绍，支持富文本HTML "changelog": "v1.0.1 - 首个版本", // string，可选，更新日志 "status": 2, // number，可选，0下架/1上架/2草稿，默认2 "sort_order": 0, // number，可选，排序权重，默认0，数字越大越靠前 "tags": "免费,开源" // string，可选，标签，逗号分隔 }
成功响应
{ "code": 0, "msg": "创建成功", "data": { "id": 1 } }
失败响应
{ "code": 1, "msg": "软件名称不能为空" }

POST {base}/software/api/status.php 修改软件状态 / 批量上下架
请求参数（JSON Body）
// 单个状态修改 { "id": 1, // number，必填，软件ID（与 ids 二选一） "status": 1 // number，必填，0=下架 / 1=上架 / 2=草稿 } // 批量上架 { "ids": [1, 2, 3], // array，必填，软件ID数组 "action": "online" // string，必填，online=上架 / offline=下架 }
成功 / 失败响应
{ "code": 0, "msg": "状态更新成功" } { "code": 0, "msg": "批量上架成功，共更新 3 条" } { "code": 1, "msg": "状态值无效" }

通用 所有接口通用 响应码说明
code 说明
code: 0请求成功
code: 1业务逻辑错误（参数错误、数据不存在等），见 msg 字段
code: 401未授权（未登录 / token 过期 / CSRF 校验失败）
HTTP 500服务器内部错误（数据库连接失败 / PHP 运行时错误）
常用 msg 示例
"软件名称不能为空" - name 字段为空
"下载地址不能为空" - download_urls 字段为空
"参数错误：缺少软件ID" - update 时未传 id
"状态值无效" - status 不在 [0,1,2] 范围内
"分类名称已存在" - 分类名重复（唯一约束）
"该分类下有 N 个软件，无法删除" - 分类下有关联软件
"未授权访问" - 未带 token 或 token 无效
