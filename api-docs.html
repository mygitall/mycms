接口总览

MYCMS API 共 41 个接口，分为搜索、核心管理、文章、软件、管理面板五大模块。

基础地址{base} 为当前部署路径，如 http://localhost/cms
请求格式Content-Type: application/json
响应格式JSON：{"code":0,"msg":"success","data":{...}}
搜索接口支持 GET/POST，已开启 CORS 跨域
文章列表/详情公开接口，无需认证
写操作接口需要管理员 Token + CSRF 校验

认证说明

Token 认证

写操作接口需要有效的管理员 Token，支持三种传递方式（优先级从高到低）：

// 方式一：Authorization 请求头（推荐） curl -H "Authorization: Bearer <token>" ... // 方式二：X-Token 请求头 curl -H "X-Token: <token>" ... // 方式三：POST Body 中的 _token 字段 curl -d '{"_token":"<token>", ...}' ...

CSRF 校验

所有状态变更操作（POST/DELETE/PUT/PATCH）需通过 CSRF 校验。两种方式二选一：

// 方式一：CSRF Cookie + Header（推荐，适用于 API 调用） // 步骤1：获取 CSRF Token（服务端会设置 Cookie） curl -c cookies.txt "{base}/api/csrf_token" // 步骤2：后续请求带上 Cookie 和 X-CSRF-Token 头 curl -b cookies.txt -H "X-CSRF-Token: <csrf_token>" ... // 方式二：Referer/Origin 同源校验（适用于浏览器表单） // 请求头 Referer 或 Origin 与 Host 一致时自动放行

频率限制

各接口有独立的频率限制，超限返回 429 状态码 + Retry-After 响应头。封禁记录存储在 storage/runtime/ 目录。

统一响应格式

{ "code": 0, // 0=成功, 400=参数错误, 401=未登录, 403=CSRF/权限不足 "msg": "success", // 429=频率限制, 500=服务器错误 "data": { ... } // 响应数据，失败时为 null }

搜索模块

GET/POST {base}/search/api/list 全站搜索 公开CORS
请求参数
{ "keyword": "PHP教程", // string，必填，≤100字符 "source": "article", // string，可选：article / software / 空=全部 "page": 1, // number，可选，默认 1 "page_size": 10, // number，可选，1-100，默认 10 "sort_by": "relevance" // string，可选：relevance / created / views }
成功响应
{ "code": 0, "msg": "success", "data": { "list": [ { "type": "article", "id": 1, "title": "PHP入门", "category": "PHP", "tags": "教程,入门", "author": "admin", "views": 100, "url": "http://localhost/cms/article/p/1", "date": "2026-05-15 10:00:00", "content": "PHP入门教程，从零开始...", "subtitle": "PHP · 教程,入门", "relevance": 4 } ], "total": 42, "sources": ["article","software"], "page": 1, "page_size": 10, "total_pages": 5, "base_path": "/cms", "base_url": "http://localhost/cms" } }
Relevance 评分规则
title 命中：+3 分 其他字段命中（content / category / tags / name / description）：+1 分/字段 多字段同时命中时分数累加 默认排序：relevance DESC, created_at DESC
外部调用示例（JavaScript）
fetch('{base}/search/api/list', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ keyword: 'PHP', page: 1, page_size: 10 }) }).then(r => r.json()).then(data => { // data.list[].url 已是完整 URL，可直接使用 data.list.forEach(item => console.log(item.title, item.url)); });
失败响应
{ "code": 400, "msg": "关键词长度不能超过100个字符", "data": null } { "code": 400, "msg": "无效的数据源类型: xxx", "data": null }

请求参数
{ "action": "list", // string，必填：list / delete / clear "_token": "adminToken",// string，必填，管理员 Token "page": 1, // number，list 时必填 "page_size": 20, // number，可选，默认 20 "keyword": "PHP", // string，可选，关键词过滤 "id": 1 // number，delete 时必填 }
成功响应
{ "code": 0, "msg": "success", "data": { "list": [{ "id": 1, "keyword": "PHP", "search_count": 5, "last_searched_at": "..." }], "total": 42, "page": 1, "page_size": 20, "total_pages": 3, "stats": { "total": 100, "today": 12, "top_keyword": "PHP" } } }

核心 API

POST {base}/api/login 用户登录 公开
请求参数
{ "username": "admin", // string，必填 "password": "admin123", // string，必填 "device": "iPhone15", // string，可选，设备标识 "remember": true // bool，可选，记住登录（Token 有效期 30 天） }
成功响应
{ "code": 0, "msg": "登录成功", "data": { "token": "62292df0...", "user_id": 1, "username": "admin", "expires_at": "2026-05-22 10:05:37", "expires_in": 604800 } }
限流与安全
// 同一 IP 5 次/5 分钟内登录失败即封禁 30 分钟 // CSRF 校验：支持 Referer 同源自动放行 // 密码比对采用恒定时间比较，防时序攻击

成功响应
{ "code": 0, "msg": "注册成功", "data": { "token": "...", "user_id": 8, "username": "newuser", "expires_at": "..." } }
限流
// 同一 IP 每小时最多注册 10 次，超限封禁 30 分钟
