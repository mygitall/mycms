MYCMS 标签占位符参考文档

标签语法规则

[--标签开始标记。所有标签必须以 [-- 开头
--]标签结束标记。所有标签必须以 --] 结尾
loop:循环块开始。[--loop:标签名(参数)--] 表示开始一个循环，会遍历数据并重复输出中间的HTML
/loop:循环块结束。[--/loop:标签名--] 表示循环到此结束
(key=val)传递参数。括号内用逗号分隔键值对，如 (num=10,cat=技术)
[--xxx--]简单标签。直接替换为对应内容（如站点名、年份等）
循环内[--字段--]数据字段。在循环块内部使用，输出当前这条数据的某个字段值
[--include(name=xxx)--]引入公共模板片段。包含 templates/v4/xxx.html 的内容，并递归渲染其中的标签

示例：软件列表循环详解

[--loop:software(num=50)--] ← 开始循环，传参数 num=50 <div> <a href="[--url--]"> ← [--url--] 输出当前软件的详情页URL <strong>[--name--]</strong> ← [--name--] 输出软件名称 </a> <span>v[--version--]</span> ← [--version--] 输出版本号 <span>[--category--]</span> ← [--category--] 输出分类名 <p>[--summary--]</p> ← [--summary--] 输出简介（前80字） </div> [--/loop:software--] ← 结束循环
这段代码的作用：从数据库 software 表取出最多 50 条已发布软件，每条生成一个 <div> 块， 其中名称、版本、分类、简介自动替换为实际数据。
快速导航： 系统标签 内容标签 分类标签 栏目标签 导航标签

系统标签

site_name — 站点名称
[--site_name--]
输出站点名称。从系统配置表读取，未设置时默认显示 "MYCMS"。
无参数
site_url — 站点URL
[--site_url--]
输出完整的站点URL（协议 + 域名 + 路径前缀）。如 http://localhost 或 https://example.com/cms
无参数
current_year — 当前年份
[--current_year--]
输出当前四位年份数字，如 2026。常用于页脚版权信息。
无参数
current_date — 当前日期
[--current_date--] [--current_date(format=Y年m月d日)--]
输出当前日期。默认格式 YYYY-MM-DD，可通过 format 参数自定义。
参数：format — PHP date() 格式字符串，如 Y-m-d / Y年m月d日 / m/d/Y
config — 读取配置
[--config(key=site_name)--]
读取系统配置表中任意配置项的值。
参数：key — 配置项的键名（必填）
search_form — 搜索表单
[--search_form--]
输出一个搜索表单HTML（GET方式提交到 /search）。含文本框和提交按钮。
无参数
login_form — 登录表单
[--login_form--]
输出一个登录表单HTML（POST方式提交到 /login），含用户名、密码字段和CSRF保护。
无参数
include — 引入公共模板
[--include(name=header)--] [--include(name=footer)--]
引入 templates/v4/ 下的公共模板文件。常用于引入公共头部、公共底部。引入的文件内仍然可以使用占位符标签，include 支持递归渲染。
参数：name=文件名（不含.html后缀，只允许字母数字和连字符）
column_info — 当前栏目名
[--column_info--]
读取URL参数 ?col=ID，返回对应栏目的名称。用于共享模板 article-list.html 中动态显示栏目标题。
无参数，自动从URL读取

内容标签（文章 + 软件）

articles — 文章列表循环
[--loop:articles(num=10,cat=技术,sort=view_count,order=DESC)--] <a href="[--url--]">[--title--]</a> <span>[--category--]</span> <span>[--published_at--]</span> <span>[--view_count--] 次阅读</span> <p>[--summary--]</p> [--/loop:articles--]
遍历已发布文章列表。每条文章循环输出中间HTML一次。
参数：num=数量（默认10） cat=分类筛选 sort=排序字段(view_count/created_at/published_at) order=DESC/ASC
可用变量： [--title--] 标题   [--url--] 详情页URL   [--summary--] 内容摘要（前120字）   [--category--] 分类   [--view_count--] 浏览量   [--published_at--] 发布时间   [--cover_image--] 封面图URL   [--author_name--] 作者
article_list — 文章列表（直接输出HTML）
[--article_list(num=10)--] [--article_list(num=5,cat=技术,sort=view_count)--]
直接输出带 <ul><li> 结构的文章列表HTML，无需写循环体。
参数：同 articles 标签
article_detail — 文章详情
[--loop:article_detail--] <h1>[--title--]</h1> <div>[--content--]</div> <span>[--author_name--]</span> <span>[--published_at--]</span> <span>[--view_count--] 次阅读</span> [--/loop:article_detail--]
用于文章详情页（detail.html）。自动读取URL中的文章ID，返回该文章的全部字段。
无需参数，自动从当前页面URL获取文章ID。也可手动指定：id=文章ID
可用变量： [--title--]   [--content--] 完整正文   [--category--]   [--tags--]   [--view_count--]   [--author_name--]   [--published_at--]   [--cover_image--]   [--url--]
related_articles — 相关文章推荐
[--loop:related_articles(num=5)--] <li><a href="[--url--]">[--title--]</a></li> [--/loop:related_articles--]
根据当前文章的分类，推荐同分类下的其他文章。自动排除当前文章自身。
参数：num=数量（默认5）。自动读取当前页面文章ID，也可手动指定 id=文章ID
可用变量： [--title--]   [--url--]   [--summary--]   [--view_count--]   [--category--]   [--cover_image--]
software — 软件列表循环
[--loop:software(num=6)--] <a href="[--url--]"><strong>[--name--]</strong></a> <span>v[--version--]</span> <span>[--category--]</span> <p>[--summary--]</p> <span>[--view_count--] 次浏览</span> <span>[--download_count--] 次下载</span> [--/loop:software--]
遍历已发布软件列表。每条软件循环输出中间HTML一次。
参数：num=数量（默认6）
可用变量： [--name--] 软件名   [--url--] 详情URL   [--summary--] 简介(前80字)   [--version--] 版本号   [--category--] 分类   [--view_count--] 浏览量   [--download_count--] 下载量   [--download_urls--] 下载地址
software_list — 软件列表（直接输出HTML）
[--software_list(num=6)--]
直接输出带HTML结构的软件列表，无需写循环体。
参数：num=数量
software_detail — 软件详情
[--loop:software_detail--] <h1>[--name--]</h1> <span>v[--version--]</span> <span>[--category--]</span> <div>[--description--]</div> <a href="[--download_urls--]">下载</a> <span>[--view_count--] 浏览</span> <span>[--download_count--] 下载</span> [--/loop:software_detail--]
用于软件详情页（software-detail.html）。自动读取URL中的软件ID。
无需参数，自动从URL获取软件ID。也可手动指定：id=软件ID
可用变量： [--name--]   [--url--]   [--version--]   [--category--]   [--description--] 完整描述   [--download_urls--] 下载地址   [--view_count--]   [--download_count--]

分类标签

categories — 分类列表循环
[--loop:categories--] <a href="[--url--]">[--name--]（[--cnt--]篇）</a> [--/loop:categories--]
遍历所有文章分类（去重）。按文章数量降序排列，最多20个。
无参数
可用变量： [--name--] 分类名   [--url--] 分类链接(/list?cat=xxx)   [--cnt--] 该分类下的文章数量
category_nav — 分类导航（直接输出HTML）
[--category_nav--]
直接输出带 <nav><ul><li> 结构的分类导航HTML，无需写循环体。
无参数

栏目标签

columns — 栏目列表循环
[--loop:columns(pid=0)--] <a href="[--url--]">[--name--]</a> <span>[--type--]</span> [--/loop:columns--]
遍历指定父栏目下的子栏目。pid=0 为顶级栏目，pid=栏目ID 为该栏目下的子栏目。不自动递归。
参数：pid=父栏目ID（默认0=顶级）
可用变量： [--id--] 栏目ID   [--name--] 栏目名称   [--url--] 栏目链接   [--type--] 类型(list/page/link)   [--parent_id--] 父栏目ID
column_tree — 栏目树（自动递归所有层级）
[--column_tree--] [--column_tree(pid=0)--] [--column_tree(pid=0,class=nav-list)--]
自动递归渲染完整栏目树（含所有子栏目）。输出 <ul><li> 结构的HTML。适合导航菜单。
参数：pid=起始父ID(默认0=顶级) class=ul的CSS类名

导航标签

breadcrumb — 面包屑导航
[--breadcrumb--]
根据当前URL自动生成面包屑导航路径。如：首页 / 文章 / 标题
无参数
pagination — 分页导航
[--pagination(total=100,per_page=10,page=1,url=?page=)--]
输出分页导航条。含首页/上一页/页码/下一页/末页 + 总记录数。当前页高亮。
参数： total=总记录数   per_page=每页条数(默认10)   page=当前页码(默认1)   url=链接前缀(如 ?page= 或 /list?page=)
carousel — 首页轮播数据
[--carousel(num=5)--]
返回JSON格式的轮播数据数组。每个元素含 title/category/url/cover/desc 字段。用于JS渲染轮播图。
参数：num=数量（默认5）

使用说明

在模板中使用标签

1. 标签只能用于模板HTML文件（templates/v4/*.html），不能用于PHP文件。
2. 标签在服务端渲染，浏览器收到的已经是替换后的HTML，对SEO友好。
3. 模板修改后自动重新编译，无需手动清理缓存。
4. 标签遵循「编译一次，缓存复用」机制，性能接近原生PHP include。
5. 循环内可嵌套循环（如分类循环内嵌套文章循环），每层变量独立不冲突。

调试技巧

- 如果标签不显示内容，先检查数据是否存在（后台确认有已发布的文章/软件）。
- 查看页面源代码（Ctrl+U）确认标签是否已被替换。
- 标签语法错误（如拼写错误）不会报错，只会输出空内容。
- 循环标签闭标记必须严格匹配：[--/loop:标签名--] 中的标签名要和开标记一致。
