搜索接口文档

全站搜索功能接口说明，包含文章和软件的全文搜索能力 · v1.0.1

接口总览
接口名称 请求地址 请求方式 说明
全站搜索 /search/api/list GET / POST 搜索文章和软件，支持关键词、分页、排序
搜索历史管理 /admin/api/search_history POST 管理员查看、删除、清空搜索历史
全站搜索 GET /search/api/list
请求参数（Query String / JSON Body）
参数名 类型 必填 说明
keyword string 是 搜索关键词
source string 否 数据源：article / software / 空（全部）
page int 否 页码，默认 1
page_size int 否 每页条数，默认 10，最大 100
sort_by string 否 排序方式：relevance / created / views，默认 relevance
请求示例
{ "keyword": "PHP教程", "source": "article", "page": 1, "page_size": 10, "sort_by": "relevance" }
成功响应
{ "code": 0, "msg": "success", "data": { "list": [ { "type": "article", "id": 1, "title": "PHP入门教程", "url": "/article/p/1" }, { "type": "software", "id": 2, "name": "PHPStorm", "url": "/software/p/2" } ], "total": 42, "sources": ["article", "software"], "page": 1, "page_size": 10, "total_pages": 5 } }
失败响应
{ "code": 400, "msg": "keyword 参数不能为空", "data": null }
搜索历史管理 POST /admin/api/search_history
请求参数（JSON Body）
参数名 类型 必填 说明
action string 是 操作类型：list / delete / clear
page int 否 list 时必填，页码
page_size int 否 每页条数，默认 20
keyword string 否 list 时用于关键词过滤
id int 否 delete 时必填，历史记录 ID
请求示例
{ "action": "list", "page": 1, "page_size": 20, "keyword": "PHP" }
成功响应
{ "code": 0, "msg": "success", "data": { "list": [ { "id": 1, "keyword": "PHP", "search_count": 5, "created_at": "2026-05-15 10:00:00" } ], "total": 42, "stats": { "total": 100, "today": 12, "top_keyword": "PHP" } } }
失败响应
{ "code": 403, "msg": "权限不足", "data": null }
